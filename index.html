🔍 4. High-Level Scope

▼

✅ In Scope for Semester Project

  • A responsive website with a clean, professional design
  • A home page with an introduction and summary about me
  • A portfolio section showing 2 data analytics projects with visuals and short explanations
  • A resume page with a downloadable PDF option
  • A contact form for messages
  • A section showing relevant certifications (Google Data Analytics)

🗺️ Roadmap Items (≤6 Months)

  • Interactive dashboards using Tableau, Power BI, or Python
  • Links to LinkedIn and GitHub for automatic updates
  • Updates/new project additions
  • A custom domain name

❌ Out of Scope

  • Build a big company-level data system
  • Set up automatic data pipelines that refresh by themselves
  • Add advanced AI features (like chatbots or recommendations)
  • Create a mobile app version of the portfolio
  • Support multiple languages

⚙️ 6. Project Methodology

▼

Explanation:

My requirements are well-defined and unlikely to change → Predictive

For my project, predictive is a good methodology as it is straightforward and works with my schedule throughout the semester. It defines a start and an end which allows me to stay on task, and lets me comfortably build my portfolio piece by piece.

🛠️ 7. Project Tool Selection

▼

Explanation:

I chose Microsoft Project (Planner)

I chose Microsoft Project (Planner) as my tool for this project because it is well-suited for predictive project management. My personal portfolio project has a fixed end date at the end of the semester, and Microsoft Project allows me to create a detailed schedule with clear dependencies, milestones, and deadlines. Its calendar functionality helps visualize the timeline, ensuring tasks are completed in sequence and on time. This level of structure and control supports the predictive methodology's focus on upfront planning and minimizing risks.
